### Expressões comparativas

|Operador|Significado     |
|--------|----------------|
| >      | maior          |
| <      | menor          |
| >=     | maior ou igual |
| <=     | menor ou igual |
| ==     | igual          |
| !=     | diferente      |

### Expressões Lógicas

|Operador|Significado   | Obs                      |
|--------|--------------|--------------------------|
| &&     | E            |Todas as condições V      |
| ||     | OU           |Pelo menos uma condição V |
| !      | NÃO          |Inverte a condição        |
