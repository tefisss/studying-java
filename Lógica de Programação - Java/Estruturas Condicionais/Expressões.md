### Expressões comparativas

|Operador|Significado     |
|--------|----------------|
| >      | maior          |
| <      | menor          |
| >=     | maior ou igual |
| <=     | menor ou igual |
| ==     | igual          |
| !=     | diferente      |

### Expressões Lógicas

- && = E (TODAS as condições são verdadeiras)
- || = OU (PELO MENOS UMA condição é verdadeira)
- ! = NÃO (Inverte a condição)

### Operadores de atribuição cumulativa
|Operadores de Atribuição Cumulativa|
|--------|--------------------------|
|a += b; | a = a + b;               |
|a -= b; | a = a - b;               |
|a *= b; | a = a * b;               |
|a /= b; | a = a / b;               |
|a %= b; | a = a % b;               |
